About 'Flying Through Time'

The project will be an interactive globe visualizing the different factors affecting the evolution of aviation over time. The domain/URL will be www.flyingthroughtime.org. The website will be centralized around a globe including our three major airports. There will be an area available to the user allowing a choice of what will be visualized (i.e. average altitudes, number of flights, specific airport, average number of passengers per plane, etc.). If the user is to select number of flights for a particular airport, it would be visualized by an airplane icon (every airplane icon represents 1,000 flights). If the user is to select average altitude, it would be visualized by the distance away from the globe; cumulitive altitude would be displayed by stacking the average altitude distances on top of one another. If the user is to select average number of passengers per plane, the airplane icon would be enlarged or shrunk. Within all of these options, scrolling over the icon would allow the historical data to emerge. There will be a time slider on the bottom, which would demonstrate the change of these variables over time.
